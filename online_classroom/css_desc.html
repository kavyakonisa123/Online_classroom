About the Course

This is for beginners to learn CSS.

  • Duration: 5 weeks
  • Time commitment: 2-3 hours per week
  • Prerequisites: HTML

What You'll Learn

  • The fundamentals of CSS and Box Model
  • Display and Positioning
  • Colors and Typography
  • Simple web page using HTML and CSS
